# Библиотеки для そうです / Sodesu

Дополнительные словарные библиотеки в формате CSV, которые можно докачать и импортировать в приложение.

- `genki/` — материалы по учебникам Genki 1, Genki 2 и «Полезные материалы».
- `jlpt/` — библиотеки по уровням JLPT (N5–N1), основанные на данных [jisho.org](https://jisho.org).

## Как установить
**Настройки → Библиотеки → ⬆️ Импорт библиотеки (.csv)** → выберите скачанный файл.

## Формат
`type,group,kanji,kana,translation,pattern` — см. подробности в `jlpt/README.md`.
Несколько значений перевода можно разделять `;`.
