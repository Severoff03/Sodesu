# Библиотеки JLPT (на базе jisho.org)

Словарные библиотеки по уровням **JLPT N5–N1**. Данные основаны на [jisho.org](https://jisho.org).

## Файлы
- `Jisho-N5.csv` … `Jisho-N1.csv` — по одному файлу на уровень.

## Как установить
В приложении: **Настройки → Библиотеки → ⬆️ Импорт библиотеки (.csv)** → выберите нужный файл.
Библиотека появится в источниках словаря и деки.

## Формат
```
type,group,kanji,kana,translation_ru,translation_en,pattern
```
- `type` — `word` / `kanji` / `grammar`;
- `group` — занятие или тема (для JLPT — уровень, напр. `Jisho-N5`);
- `kanji`, `kana` — написание;
- `translation_ru`, `translation_en` — перевод (несколько значений можно разделять `;`);
- `pattern` — пример/конструкция (опционально).

> Примечание: русские переводы заполнены не для всех слов — это шаблоны на базе jisho,
> которые будут дополняться. Английский перевод присутствует у большинства записей.
