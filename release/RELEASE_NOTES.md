# Sodesu 0.9 (beta)

Публичный бета-релиз.

- Карточки с интервальным повторением (кандзи / слова / грамматика)
- Офлайн-словарь (поиск по кандзи, кане, английскому, русскому)
- Грамматика с объяснениями
- Тесты и JLPT-пробники N5–N3
- Темы и свои фоны
- Перенос прогресса между устройствами

Полный список изменений — в [PATCHNOTES.md](PATCHNOTES.md).

**Установка:** скачай `Sodesu-0.9-beta.apk`, разреши установку из неизвестных источников, открой. Интернет не нужен.

> ВАЖНО: бета. Библиотеки пока покрывают уровни N5–N4. Бесплатно, без рекламы.
